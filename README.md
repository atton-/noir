# Noir

Noir is utility commands for atton.


## Installation

    $ gem install noir

## Usage

    $ noir <command> [sub-command...] [args-for-command]

## Enable Completion

    Add this code in .zshrc

    ``` if which noir  >& /dev/null; then eval "$(noir init zsh)"; fi ```
